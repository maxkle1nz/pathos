# Code of Conduct

PATHOS exists to make human/agent collaboration more honest, recoverable, and
safe. Treat contributors and users with clarity and respect.

Expected behavior:

- be precise;
- preserve proof;
- disagree without contempt;
- protect secrets;
- avoid unsupported claims.

Unacceptable behavior:

- harassment;
- publishing private credentials;
- misrepresenting unverified capability;
- intentionally destructive changes.

Report concerns through the repository issue tracker or maintainer contact path.
