# Public Profile Widgets

These widgets can be used in a GitHub profile README, project README, or public
landing page. They are optional: too many widgets can make a serious project look
noisy, so prefer a compact top badge row and put the rest behind a collapsible
section.

Replace `maxkle1nz` and `maxkle1nz/pathos` if you fork the project.

## Widget Postman Rule

Use widgets as trust signals, not decoration. If the README becomes harder to
understand, collapse the widget wall or move it to this file.
